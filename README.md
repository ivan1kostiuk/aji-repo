## Quiz Länder

### Beskrivning

Quiz-spel där man kan gissa vilken flagga hör till vilket land, vilket land som har vilken form, vilket land som har större befolkning utav två val, etc. Använder publika API's för information om länderna. 

### MVP

Ett lättare geografispel som kan köras i terminalen.

Ex: Vilket av tre länder ligger i Europa, vilket av tre länder har störst befolkning.

### Teknisk Specifikation

Språk: JavaScript + HTML/CSS

Frameworks: Jest (för tester)

Verktyg: VS Code
